GitHub User Finder

Автор:Ахмадуллина Ляйсан

Описание

«GitHub User Finder» — это десктопное GUI-приложение для поиска пользователей GitHub.
Программа позволяет находить пользователей по логину, просматривать их основную информацию (аватар, имя, репозитории, подписчиков и т. д.), а также добавлять пользователей в избранное. Список избранных сохраняется локально в JSON-файл, благодаря чему он не теряется после закрытия приложения.

Приложение написано с использованием Python и tkinter для графического интерфейса, а для работы с сетью используется библиотека requests.

Как использовать GitHub API

1. Получение данных пользователя
   Приложение отправляет GET-запрос к эндпоинту GitHub API:
   https://api.github.com/users/{username}
   В ответ приходит JSON-объект с информацией о пользователе (логин, имя, аватар, количество репозиториев, подписчиков и т. д.).
2. Поиск пользователя
   Если необходимо найти нескольких пользователей по строке поиска, используется эндпоинт:
   https://api.github.com/search/users?q={query}
   Приложение отображает список найденных пользователей.
3. Обработка ошибок
   · При пустом поле поиска — выводится предупреждение.
   · Если пользователь не найден (статус 404) — показывается сообщение об ошибке.
   · При превышении лимита запросов (статус 403) — программа уведомляет о необходимости подождать.

Требования для работы API:

· Библиотека requests.
· Интернет-соединение.
· (Опционально) Токен авторизации GitHub для увеличения лимита запросов.

Примеры использования

Запуск приложения

```bash
git clone <ваш_репозиторий>
cd github-user-finder
pip install -r requirements.txt
python main.py
```

Пример поиска

1. В поле ввода введите octocat.
2. Нажмите «Поиск».
3. Отобразится информация о пользователе Octocat (GitHub-маскот).
4. Нажмите «Добавить в избранное» — пользователь сохранится в favorites.json.
